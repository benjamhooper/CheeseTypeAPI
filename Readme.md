# CheeseTypeAPI

ASP.NET Core API with MongoDB

1. Install Docker & docker-compose
2. Run `docker-compose up -d` within the directory

Apart of the CheeseBiz Microservice Architecture

`GET /api/CheeseType`
`POST /api/CheeseType`

> Example responses
